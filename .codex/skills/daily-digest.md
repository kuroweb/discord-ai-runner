# daily-digest

## Goal
当日の主要なAIニュースを日本語で要約し、Discord向けに読みやすいデイリーダイジェストを作成する。

## Workflow
1. WebSearch / WebFetch を使って、当日〜直近24時間のAI関連ニュースを収集する。
2. 同じ事象の重複記事をまとめ、一次情報（公式発表・企業ブログ）を優先する。
3. 重要度順に3〜7件へ絞り、各項目を「何が起きたか / なぜ重要か」で要約する。
4. 不確実な情報は推測せず、確認できた事実のみを書く。

## Output format
- 1行目: タイトル用の短い要約（平文・20文字程度）
- 空行
- 本文: Markdown
  - `## 今日のハイライト`
  - 箇条書きでニュース要点
  - `## 詳細`
  - 各ニュースを `### 見出し` + 2〜4行で要約
  - `## ソース`
  - 参照URLを箇条書き

## Constraints
- 日本語で書く。
- 誇張や煽り表現は避け、事実ベースで記述する。
- 日付は可能な限り具体的に書く。
